Materials

Introduction to Swift & iOS, 2016

This workshop is an introduction to Swift and iOS development. Participants learnt:

  • Learning the essentials of Swift, Apple's new programming language
  • Building a simple iOS App with UI tied to backend logic using Model-View-Controller pattern

Software Used

Participants are required to download and use the following pieces of software:

  • OS X (El Capitan)
  • Xcode 7.2.1
