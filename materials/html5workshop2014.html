Introduction to HTML5/CSS3, 2014

This workshop is a basic introduction to HTML and CSS. Participants learnt:

  • Basic HTML
  • Basic CSS
  • Putting it all together in a (beautiful!) personal web page
  • Twitter's Bootstrap CSS framework
  • Some HTML5 elements

Software Used

Participants are required to download and use the following pieces of software:

  • Google Chrome
  • Sublime Text 2
